# Claude Project

这是 Claude Code 的默认工作目录，用于存放各类 AI 模型测试项目。

## 项目结构

```
ClaudeProject/
├── gemma-4_test/           # Gemma-4 系列模型显存测试
└── gold_price_pusher/      # 黄金价格定时推送脚本
```

## 工作规则

1. 新项目以文件夹形式存放在根目录下
2. 每个项目包含独立的 README.md 和测试代码
3. 使用 Git 进行版本控制，推送到 GitHub 仓库
